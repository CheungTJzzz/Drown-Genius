Deterministic Optimal Growth Model¶

  • Consider the following social planner's problem: $$ \max_{c_t,k_{t+1}} \sum_{t=0}^{\infty} \beta^t \ln c_t, $$ s.t. $c_t+k_{t+1}=k_t^{\alpha}$, and $k_0>0$ is given.

Bellman Equation¶

  • The recursive formulation is given by $$ v(k_t)=\max_{c_t,k_{t+1}} [\ln c_t + \beta v(k_{t+1})], $$ s.t. $c_t+k_{t+1}=k_t^{\alpha}$, and $k_0>0$ is given.

  • Or $$ v(k_t)=\max_{k_{t+1}} [\ln (k_t^\alpha - k_{t+1}) + \beta v(k_{t+1})] $$

Solution¶

  • FOC and envelope theorem $$ \beta v'(k_{t+1}) = \frac{1}{c_t}, v'(k_t)=\frac{\alpha k_t^{\alpha-1}}{c_t} $$

  • Euler equation $$ \frac{c_{t+1}}{c_t}=\alpha \beta k_{t+1}^{\alpha-1} $$

  • Steady state of $k$ $$ k^*=(\alpha \beta)^{\frac{1}{1-\alpha}} $$

  • Analytical solution $$ \begin{split} v(k) &= \frac{\ln(1-\alpha \beta)}{1-\beta} + \frac{\alpha \beta \ln(\alpha \beta)}{(1-\beta) (1-\alpha \beta)} + \frac{\alpha}{1-\alpha \beta} \ln k \\ k_{t+1} &= \alpha \beta k_t^\alpha \end{split} $$

Numeric Method: Value function iteration¶

  1. Take an arbitary intial guess of $v$.
  2. Obtain an update $Tv$: $Tv(k)=\max [{\ln(c)+\beta v(k)}]$.
  3. Check whether $v=Tv$.
    • Stop if $Tv$ is approximately equal to $v$.
    • Otherwise set $v=Tv$ and go back to step 2.

Calibration¶

$$ \begin{split} \beta&=0.95, \\ \alpha&=1/3, \\ k&\in [\frac{1}{2}k^*, \frac{3}{2} k^*]. \end{split} $$

Code¶

# import

import timeit
import numpy as np
import matplotlib.pyplot as plt

Set model parameters¶

beta = 0.95
alpha = 1 / 3
kss = (alpha * beta) ** (1 / (1 - alpha))
kmin = 0.5 * kss
kmax = 1.5 * kss
n = 100
kgrid = np.linspace(kmin, kmax, n)

Procedure Oriented: Step 1¶

# step 1

V0 = np.zeros(n)

Procedure Oriented: Step 2¶

def V_update(V):
    V_new = np.zeros(n)  # initialize updated value function and policy function
    g_new = np.zeros(n)
    for i in range(n):  # loop over grid points
        k = kgrid[i]
        k_bound = k**alpha  # keep non-zero consumption
        V_max = -np.inf  # initialize maximum value
        for j in range(n):  # loop over possible k_next
            k_next = kgrid[j]
            V_next = V[j]
            if k_next < k_bound:  # check feasibility
                c = k**alpha - k_next
                V_current = np.log(c)+beta*V_next
                if V_current > V_max:
                    V_max = V_current
                    g_k = k_next
        V_new[i] = V_max  # save
        g_new[i] = g_k
    return V_new, g_new

Procedure Oriented: Step 3¶

def V_iteration(V_initial):
    V = V_initial
    error = np.inf
    count = 0
    max_iter = 1000
    print_skip = 50
    tol = 1e-6
    while count < max_iter and error > tol:
        V_new, g_new = V_update(V)
        error = np.max(np.abs(V_new - V))
        V = V_new
        count = count + 1
        if count % print_skip == 0:
            print(f"Error at iteration {count} is {error}.")
    if error > tol:
        print("Failed to converge!")
    else:
        print(f"\nConverged in {count} iterations.")
    return V_new, g_new
